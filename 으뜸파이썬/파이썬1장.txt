인터프리터라는 프로그램이 hello.py라는 프로그램을 기계어 명령어로 변환하여 컴퓨터에서 실행


프로그래밍 언어
- 컴퓨터 시스템을 구동시키는 소프트웨어를 만들기 위한 형식을 제대로 갖춘 언어
- 자동차나 프린터 등의 미리 틀이 정해진 기계에서 제한된 목적과 용도로 사용되는 프로그래밍 언어는 수행 속도가 중요하므로 C언어를 주로 사용
- 웹 서비스를 위해서는 HTML과 같은 마크업 언어와 JavaScript, PHP와 같은 언어가 많이 사용됨
- 파이썬은 강력하면서도 배우기 쉬움


소스 코드 : 프로그래밍 언어로 작성된 명령어들의 목록
소스 파일 : 소스 코드가 저장된 파일(***.py)


컴파일과 인터프리터

- 컴파일 방식
  - 프로그램 명령어를 기계어로 번역한 후 이 기계어를 실행하는 방식
  - C, C++, 파스칼 언어 등
  - 일반적인 경우 속도가 더 빠르다

- 인터프리터
  - 프로그램 명령어를 한 번에 한 줄씩 읽어 번역한 후 바로 실행
  - 파이썬, BASIC 언어 등
  - 실행 시간이 느리다
