제어문 : 프로그램의 흐름을 제어하는 역할, 조건문(if-else문), 반복문(for, while문), 반복문의 흐름 변경(break, continue)

순차적 구조 : 먼저 나타나는 코드가 먼저 실행되는 구조

조건문 : 조건에 따라 실행할 문장을 선택하는 구조

파이썬의 들여쓰기 : 파이썬은 들여쓰기가 블록을 표현(매우 중요한 의미) - C나 JAVA는 {}가 블록을 표시

break와 continue : 반복문을 제어하는 키워드
break->else : 블록을 실행하지 않고 반복을 종료
continue : 반복문 루프 내의 나머지 실행 문장을 건너뛰고 반복 루프를 실행
